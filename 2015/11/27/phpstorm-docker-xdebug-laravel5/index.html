Phpstorm Docker Xdebug Laravel5

Docker 镜像下载

  • docker pull wsmdr1986/php5.6
  • docker pull wsmdr1986/nginx

    Laravel 下载

  • composer create-project laravel/laravel --prefer-dist

开始

  • docker run -tid --name=php -v /Users/wsmdr/sites/laravel:/app wsmdr1986/php5.6
  • docker run -tid --name=www -v /Users/wsmdr/mydocker/conf_laravel_php56:/etc/nginx/conf.d --link php:php --volumes-from php -p 8001:80 wsmdr1986/nginx
  • conf_laravel_php56 文件夹里为default.conf代码
  • 进入php container配置xdebug
  • 浏览器种cookie
  • Phpstorm 常规配置
